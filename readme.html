Wstęp:

Witam, nazywam się Szymon Perlicki, a to jest dokumentacja języka Monty.
Monty jest całkowicie nowym, statycznie typowanym i interpretowanym językiem programowania, który napisałem całkowicie sam.
Z użyciem języka Monty można bardzo łatwo i szybko dokonywać obliczeń na ogromnych liczbach z dużą dokładnością.
Programy napisane w tym Montym powinny być zapisywane z rozszerzeniem ".mt".
Do uruchomienia programu, który chciałbym wam zaprezentować potrzebna jest Java co najmniej w wersji 11.0.2, aby sprawdzić, która wersja Javy znajduje się na komputerze i czy jest w ogóle zainstalowana na komputerze należy wpisać w konsoli:

java --version

Kiedy jest już zainstalowana Java w wersji 11.0.2 i jest się w folderze z interpreterem, aby uruchomić program o nazwie "program.mt", w folderze nazywającym się "examples", który jest w tym samym folderze gdzie interpreter trzeba wpisać:

java -jar Monty.jar examples/program.mt

Jeśli chcesz napisać jakiś program w języku Monty zapraszam do podstrony "Jak zacząć?", a potem do dokumentacji biblioteki standardowej języka Monty w podstronie "Biblioteka Standardowa".

Gdzie jest Monty?:

Interpreter Monty ("Monty.jar") znajdziesz w folderze o nazwie "bin", który jest w folderze wcześniej ("..").
W katalogu z interpreterem umiejscowiłem też folder "examples", w którym znajdują się przykłady prostych programów napisanych w języku Monty, i które można uruchomić za pomocą wyżej podanej komendy. Folder z kodem źródłowym znajduje się w tym samym folderze co "bin" i nazywa się "src". Kopia projektu znajduje się też na githubie.
