io:

Moduł io zawiera funkcje umożliwiające obsługę wyjścia i wejścia standardowego.

input:

func string input;
Przyjmuje wejście standardowym wejściem i zwraca je.

print:

func void print any toPrint;
Wyświetla toPrint w postaci ciągu znaków.

println:

func void println any toPrint;
Wyświetla toPrint w postaci ciągu znaków i dodaje na koniec nową linijkę.
