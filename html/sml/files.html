files:

Moduł files zawiera funkcje umożliwiające zapis i odczyt plików.

absPath:

func string absPath string path;
Zwraca ścieżkę path w formacie ścieżki absolutnej.

exists:

func boolean exists string path;
Jeśli path istnieje zwraca true, w przeciwnym wypadku zwraca false.

isAbsolute:

func boolean isAbsolute string path;
Jeśli path jest ścieżką absolutną zwraca true, w przeciwnym wypadku zwraca false.

isDir:

func boolean isDir string path;
Jeśli path jest folderem zwraca true, w przeciwnym wypadku zwraca false.
W przypadku braku dostępu do path wyświetla komunikat, miejsce błędu i wychodzi z programu.

isFile:

func boolean isFile string path;
Jeśli path jest plikiem zwraca true, w przeciwnym wypadku zwraca false.
W przypadku braku dostępu do path wyświetla komunikat, miejsce błędu i wychodzi z programu.

listDir:

func any listDir string path;
Zwraca tablicę z nazwami wszystkich plików i folderów w path, jeśli path jest plikiem zwraca pustą tablicę.

read:

func string read string path;
Zwraca zawartość pliku pod ścieżką path.
W przypadku braku dostępu do path wyświetla komunikat, miejsce błędu i wychodzi z programu.

separator:

func string separator;
Zwraca niezależny od systemu separator ścieżki plików.

write:

func void write string path, string text, boolean isAppend;
Jeśli ¬ isAppend lub plik pod ścieżką path nie istnieje to zapisuje plik pod tą ścieżką o zawartości text, w przeciwnym wypadku na koniec pliku pod ścieżką path dodaje text.
W przypadku braku dostępu do path wyświetla komunikat, miejsce błędu i wychodzi z programu.
