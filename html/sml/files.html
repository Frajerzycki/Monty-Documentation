files:

Moduł files zawiera funkcje umożliwiające zapis i odczyt plików.

read:

func string read string path;
Zwraca zawartość pliku pod ścieżką path.

write:

func string write string path, string text, boolean isAppend;
Jeśli isAppend lub plik pod ścieżką path nie istnieje to zapisuje plik pod tą ścieżką o zawartości text, w przeciwnym wypadku dodaje na koniec pliku pod ścieżką path text.
