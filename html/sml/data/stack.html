Stack:

Stack reprezentuje stos, wszystkie poniższe funkcje są metodami tej struktury i należy je wywoływać po instacji tego obiektu, po kropce.
Konstruktorem tego obiektu jest funkcja o nazwie Stack, nie przyjmuje argumentów. Zwraca pusty stos.

Stack().equals:

func boolean equals any other;
Jeśli other jest instancją Stack, other ma tą samą długość co this i wszystkie każda kolejna para elementów z obu stosów jest sobie równa zwraca true, w przeciwnym wypadku zwraca false.
Można ją wywołać operatorem ==, kiedy lewym literałem jest stos.

Stack().length:

func int length;
Zwraca ilość elementów na stosie.

Stack().peek:

func any peek;
Zwraca element z góry stosu.

Stack().pop:

func any pop;
Zwraca element z góry stosu i usuwa go.

Stack().push:

func any push any element;
Wrzuca element na górę stosu. Zwraca referencję do stosu.

Stack().toArray:

func any toArray;
Zwraca tablicę wypełnioną elementami stosu, gdzie początkiem owej tablicy jest dół stosu.

Stack().toList:

func any toList;
Zwraca listę wypełnioną elementami stosu, gdzie początkiem owej listy jest dół stosu.

Stack().toString:

func string toString;
Zwraca elementy stosu, gdzie początkiem ciągu jest dół stosu, zamienione na ciąg znaków, oddzielone przecinkami, w nawiasach kwadratowych.
W razie potrzeby wywoływana automatycznie.
