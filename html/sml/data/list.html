List:

List reprezentuje listę linkowaną, wszystkie poniższe funkcje są metodami tej struktury i należy je wywoływać po instacji tego obiektu, po kropce.
Konstruktorem tego obiektu jest funkcja o nazwie List, przyjmuje dowolną liczbę argumentów o dowolnym typie, z argumentów podanych w konstruktorze zostanie stworzona lista linkowana.

List().append:

func any append any element;
Zwiększa liczbę elementów listy o 1, ustawia miejsce pod ostatnim indeksem na element.
Zwraca referencję do listy.

List().contains:

func boolean contains any element;
Zwraca true jeśli element znajduje się w listy, w przeciwnym wypadku zwraca false.

List().equals:

func boolean equals any other;
Zwraca true jeśli other jest instancją List, other ma tą samą długość co this, other.get(i) jest równe this.get(i), dla każdego i ∈ (-1, length), w przeciwnym wypadku zwraca false.
Można ją wywołać operatorem ==, kiedy lewym literałem jest lista.

List().extend:

func any extend any otherList;
Jeśli otherList nie jest instancją List wyświetla komunikat, miejsce błędu i wychodzi z programu.
Zwiększa liczbę elementów listy o otherList.length() i uzupełnia nowo powstałe miejsca elementami listy otherList.
Zwraca referencję do listy.

List().find:

func any find any element;
Zwraca listę z indeksami, pod którymi występuje element.

List().findFirst:

func int findFirst any element;
Jeśli istnieje element o wartości równej element zwraca indeks pierwszego takiego elementu, w przeciwnym wypadku zwraca -1.

List().findLast:

func int findFirst any element;
Jeśli istnieje element o wartości równej element zwraca indeks ostatniego takiego elementu, w przeciwnym wypadku zwraca -1.

List().get:

func any get int i;
Zwraca i-ty element listy.
Jeśli lista nie posiada i-tego elementu wyświetla komunikat, miejsce błędu i wychodzi z programu.

List().head:

func any head;
Zwraca pierwszy element listy. Jeśli lista jest pust wyświetla komunikat, miejsce błędu i wychodzi z programu.

List().length:

func int length;
Zwraca liczbę elementów listy.

List().replace:

func any replace any toBeReplaced, any replacement;
Każda wartość w listy równa toBeReplaced zamieniana jest na replacement.
Zwraca referencję do listy.

List().replaceFirst:

func any replaceFirst any toBeReplaced, any replacement;
Pierwsza wartość w listy równa toBeReplaced zamieniana jest na replacement.
Zwraca referencję do listy.

List().replaceFirst:

func any replaceFirst any toBeReplaced, any replacement;
Ostatnia wartość w listy równa toBeReplaced zamieniana jest na replacement.
Zwraca referencję do listy.

List().reversed:

func any reversed;
Zwraca nową listę o długości length(), w której wartość każdego elementu z listy this o indeksie i ∈ (-1, length()) jest przypisana do elementu w nowej listy o indeksie length() - i - 1.

List().set:

func any set int i, any value;
Ustawia i-ty element listy na value.
Zwraca referencję do listy.
Jeśli lista nie posiada i-tego elementu wyświetla komunikat, miejsce błędu i wychodzi z programu.

List().sublist:

func any sublist int begin, int end;
Zwraca podlistę listy od elementu o indeksie begin włącznie do elementy o indeksie end wyłączając go.
Jeśli begin < 0 lub end> length() wyświetla komunikat, miejsce błędu i wychodzi z programu.

List().tail:

func any tail;
Zwraca listę linkowaną bez pierwszego elementu.

List().toArray:

func any toArray;
Zwraca tablicę wypełnioną elementami listy linkowanej.

List().toStack:

func any toStack;
Zwraca stos z wypełniony elementami listy.

List().toString:

func string toString;
Zwraca elementy zamienione na ciąg znaków, oddzielone przecinkami, w nawiasach kwadratowych.
W razie potrzeby wywoływana automatycznie.
