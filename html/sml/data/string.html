string:

Moduł string zawiera zbiór funkcji pomagających w użyciu ciągów znaków, string nie jest typem złożonym, a poniższe funkcje są samodzielne.

charAt:

func string charAt int i, string str;
Zwraca i-ty znak w ciągu.
Jeśli ciąg nie posiada i-tego elementu wyświetla komunikat, miejsce błędu i wychodzi z programu.

endsWith:

func boolean endsWith string str, string suffix;
Zwraca true jeśli str kończy się na suffix, w przeciwnym wypadku zwraca false.

equalsIgnoreCase:

func boolean equalsIgnoreCase string str, string toCompare;
Zwraca true jeśli str jest równy toCompare ignorując wielkość znaków, w przeciwnym wypadku zwraca false.

lowerCase:

func boolean lowerCase string str;
Zwraca str z każdą wielką literą zamienioną na małą.

replace:

func boolean replace string str, string regex, string replacement;
Zwraca str z każdym powtórzeniem pasującym do regex zamienonym na replacement.

replaceFirst:

func boolean replaceFirst string str, string regex, string replacement;
Zwraca str z pierwszym powtórzeniem pasującym do regex zamienonym na replacement.

split:

func any split string str, string regex;
Zwraca Array obiektów podzielonych wystąpieniami ciągów znaków pasujących do regex.

startsWith:

func boolean startsWith string str, string prefix;
Zwraca true jeśli str zaczyna się na suffix, w przeciwnym wypadku zwraca false.
