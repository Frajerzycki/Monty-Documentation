Array:

Array reprezentuje tablicę, wszystkie poniższe funkcje są metodami tej struktury i należy je wywoływać po instacji tego obiektu, po kropce.
Konstruktorem tego obiektu jest funkcja o nazwie Array, przyjmuje dowolną liczbę argumentów.

Array.get:

func any Array.get int i;
Zwraca i-ty element tablicy.
Jeśli obiektu nie posiada i-tego elementu wyświetla komunikat, miejsce błędu i wychodzi z programu.

Array.length:

func int Array.length;
Zwraca liczbę elementów tablicy.
