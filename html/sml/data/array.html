Array:

Array reprezentuje tablicę, wszystkie poniższe funkcje są metodami tej struktury i należy je wywoływać po instacji tego obiektu, po kropce.
Konstruktorem tego obiektu jest funkcja o nazwie Array, przyjmuje dowolną liczbę argumentów o dowolnym typie, z argumentów podanych w konstruktorze zostanie stworzona tablica.

Array().append:

func any append any element;
Zwiększa liczbę elementów tablicy o 1, ustawia miejsce pod ostatnim indeksem na element.
Zwraca referencję do tablicy.

Array().contains:

func boolean contains any element;
Zwraca true jeśli element znajduje się w tablicy, w przeciwnym wypadku zwraca false.

Array().equals:

func boolean equals any other;
Zwraca true jeśli other jest instancją Array, other ma tą samą długość co this, other.get(i) jest równe this.get(i), dla każdego i ∈ (-1, length), w przeciwnym wypadku zwraca false.
Można ją wywołać operatorem ==, kiedy lewym literałem jest tablica.

Array().extend:

func any extend any otherArray;
Jeśli otherArray nie jest instancją Array wyświetla komunikat, miejsce błędu i wychodzi z programu.
Zwiększa liczbę elementów tablicy o otherArray.length() i uzupełnia nowo powstałe miejsca elementami tablicy otherArray.
Zwraca referencję do tablicy.

Array().find:

func any find any element;
Zwraca tablicę z indeksami, pod którymi występuje element.

Array().findFirst:

func int findFirst any element;
Jeśli istnieje element o wartości równej element zwraca indeks pierwszego takiego elementu, w przeciwnym wypadku zwraca -1.

Array().findLast:

func int findFirst any element;
Jeśli istnieje element o wartości równej element zwraca indeks ostatniego takiego elementu, w przeciwnym wypadku zwraca -1.

Array().get:

func any get int i;
Zwraca i-ty element tablicy.
Jeśli tablica nie posiada i-tego elementu wyświetla komunikat, miejsce błędu i wychodzi z programu.

Array().length:

func int length;
Zwraca liczbę elementów tablicy.

Array().replace:

func any replace any toBeReplaced, any replacement;
Każda wartość w tablicy równa toBeReplaced zamieniana jest na replacement.
Zwraca referencję do tablicy.

Array().replaceFirst:

func any replaceFirst any toBeReplaced, any replacement;
Pierwsza wartość w tablicy równa toBeReplaced zamieniana jest na replacement.
Zwraca referencję do tablicy.

Array().replaceFirst:

func any replaceFirst any toBeReplaced, any replacement;
Ostatnia wartość w tablicy równa toBeReplaced zamieniana jest na replacement.
Zwraca referencję do tablicy.

Array().reversed:

func any reversed;
Zwraca nową tablicę o długości length(), w której wartość każdego elementu z tablicy this o indeksie i ∈ (-1, length()) jest przypisana do elementu w nowej tablicy o indeksie length() - i - 1.

Array().set:

func any set int i, any value;
Ustawia i-ty element tablicy na value.
Zwraca referencję do tablicy.
Jeśli tablica nie posiada i-tego elementu wyświetla komunikat, miejsce błędu i wychodzi z programu.

Array().setLength:

func any setLength int length; Ustawia długość tablicy na length, a puste miejsca wypełnia wartością void.
Zwraca referencję do listy.

Array().subarray:

func any subarray int begin, int end;
Zwraca podlistę tablicy od elementu o indeksie begin włącznie do elementy o indeksie end wyłączając go.
Jeśli begin < 0 lub end > length() wyświetla komunikat, miejsce błędu i wychodzi z programu.

Array().toList:

func any toList;
Zwraca listę linkowaną wypełnioną elementami tablicy.

Array().toStack:

func any toStack;
Zwraca stos wypełniony elementami tablicy.

Array().toString:

func string toString;
Zwraca elementy zamienione na ciąg znaków, oddzielone przecinkami, w nawiasach kwadratowych.
W razie potrzeby wywoływana automatycznie.
