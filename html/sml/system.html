system:

Moduł system umożliwia prostą komunikację z systemem.

argv:

func string argv int index;
Zwraca argument konsolowy pod indeksem index, gdzie argv(0) zwraca wpisaną ścieżkę do pliku. Jeśli index jest ujemny albo większy lub równy liczbie argumentów wyświetla komunikat, miejscę błędu i wychodzi z programu.

exit:

func void exit int status;
Wychodzi z programu ze statusem status.
